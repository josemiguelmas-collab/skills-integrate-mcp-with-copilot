# Issues sugeridos para mejoras en la UI

## 1. Crear interfaz de usuario moderna para actividades extraescolares
**Descripción:**
Implementar una interfaz web moderna y responsiva usando Next.js y Tailwind CSS para visualizar y gestionar las actividades extraescolares.
- Listar actividades disponibles
- Mostrar detalles de cada actividad
- Permitir inscripción y baja de actividades
- Adaptarse a dispositivos móviles

---

## 2. Listado de actividades
**Descripción:**
Crear un componente que muestre todas las actividades extraescolares disponibles, mostrando nombre, descripción, horario y plazas disponibles.

---

## 3. Detalle de actividad
**Descripción:**
Crear una vista o modal que muestre la información detallada de una actividad seleccionada, incluyendo participantes actuales y opción para inscribirse o darse de baja.

---

## 4. Formulario de inscripción y baja
**Descripción:**
Implementar formularios para que los estudiantes puedan inscribirse o darse de baja de una actividad, integrando la API existente.

---

## 5. Diseño responsivo y atractivo
**Descripción:**
Aplicar estilos con Tailwind CSS para asegurar que la interfaz sea atractiva y funcional en dispositivos móviles y escritorio.
